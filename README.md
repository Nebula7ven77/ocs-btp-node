## Vue d'ensemble

**7venHotel Cloud** est un système complet de gestion hôtelière (PMS) SaaS multi-tenant.

### Modules inclus

| Module | Description |
|------|------|
| 🏨 Staff Portal | Tableau de bord, réservations, planning |
| 🌐 Booking Engine | Moteur de réservation public |
| 👤 Client Portal | Portail client |
| 📱 Room Portal | QR code chambre |
| 🤖 Ouwalou AI | Assistant IA |
| 🧹 Housekeeping | Gestion ménage |
| 🔧 Maintenance | Tickets maintenance |
| 🍽 Restaurant POS | Caisse restaurant |
| 💳 Facturation | Factures et taxes |
| 📊 Analytics | KPIs hôteliers |
